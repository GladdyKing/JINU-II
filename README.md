### 🛠️ Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/GladdyKing/JINU-II
cd JINU-II

# 2. Install dependencies
npm install

# 3. Start the bot
npm start

# 4. Enter pairing code shown in terminal into:
#    WhatsApp → Linked Devices → Link a Device
```
